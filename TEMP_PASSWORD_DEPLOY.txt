
Temporary Password (Admin-issued) Setup
======================================

This build adds an admin-only temporary password flow.

What it does:
- Admin can generate a temporary password from Team Members
- The app shows the generated password immediately to the admin
- The selected user is forced to change the password on next sign in

Deploy steps:
1) Open terminal in the project folder that contains the functions directory
2) Run:
   npm install -g firebase-tools
   firebase login
   firebase use mod-checklist-report
   cd functions
   npm install
   cd ..
   firebase deploy --only functions:issueTemporaryPassword

After deploy:
- Publish the web files as usual
- Publish Firestore Rules from firestore.rules

How to use:
- Sign in as Admin
- Go to More > Team Members
- Click "Issue Temp Password"
- Give the temporary password to the employee
- Employee signs in with that password once
- The app forces the employee to set a new password immediately
